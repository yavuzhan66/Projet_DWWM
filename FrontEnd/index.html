Creation de sites web

La création et la conception de site web ou web design est la conception de l'interface web :
l'architecture interactionnelle, l'organisation des pages, l'arborescence et la navigation dans un site web.

Création de publicités sur les réseaux sociaux

La notion de création publicitaire peut prendre deux sens. Elle peut désigner la démarche de création d'un message publicitaire par un créatif ou un team créatif,
mais elle peut également désigner l'objet publicitaire créé (spot, affiche, bannière internet, etc.).

SEO

Un référenceur web, aussi appelé référenceur SEO (pour Search Engine Optimization),
est un professionnel du référencement naturel.

Design UI/UX

L'UI design se rapporte donc à l'environnement graphique dans lequel évolue l'utilisateur d'un logiciel,
d'un site web ou d'une application. Le terme UX vient d'user expérience ou expérience utilisateur.
